# Compute Plan

The quick reproduction path uses compact processed source tables included in this repository and should run on a standard laptop.

Approximate requirements:

- Python 3.10 or later
- Memory: 4 GB RAM is sufficient for the quick path
- Disk: less than 500 MB for the repository and regenerated outputs
- Runtime: less than 5 minutes for model refresh and figure regeneration

The optional public-GEO rebuild downloads larger source files listed in `docs/DATA_MANIFEST.tsv`. Runtime and disk usage depend on network speed and GEO availability.
